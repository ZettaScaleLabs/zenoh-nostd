# `ryu-core`

This crate is the core of RYU-Z, a `#![no_std]`, `no_alloc` library that provides the fundamental building blocks for implementing the Zenoh protocol. It includes essential traits, types, and utilities needed for encoding and decoding Zenoh messages in a resource-constrained environment.
